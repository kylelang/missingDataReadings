The sources listed below offer an extended/advanced introduction to missing data
analysis. This list is meant to cover the basics of missing data from a
technical, mathematically rigorous perspective. Readers without a strong
background in mathematics/statistics may wish to begin with the sources listed
in ``gentle\_intro.tex''.\\
\vspace{12pt}
Most of the following sources represent either seminal
references in missing data or general overviews of the field. Consequently, this
list does not necessarily represent the latest work in missing data theory.
These sources should, however, provide a very thorough introduction to/overview
of modern missing data theory.
